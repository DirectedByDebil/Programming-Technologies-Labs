# Вторая лабораторная работа «Алгоритмы сортировки».
  
## Алгоритмическая сложность
  Для каждого алгоритма нужно на практике получить алгоритмическую сложность в лучшем, среднем и худшем случаях. 

### Лучший — уже отсортированный массив ✔
### Худший — отсортированный «наоборот» ✔
### Средний — чередуются элементы из худшего и лучшего случая: ✔
    самый маленький,
    самый большой,
    второй по счету маленький,
    второй по счету большой и так далее.
  
## Алгоритмы ✔
    пузырьковая сортировка ✔
    сортировка вставками ✔
    сортировка Шелла ✔
    быстрая сортировка (рекурсивный и итерационный способы) ✔✔
    поразрядная сортировка (MSD и LSD способы) ✔✔
  
## Графики
    Для каждого метода и для каждого из трех случаев построить графики зависимостей времени сортировки от количества элементов (до 100 тысяч штук) и сравнить с теоретическими значениями алгоритмической сложности.
